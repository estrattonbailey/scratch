# scratch
A little REPL CLI for fast local prototyping.

## Usage
Start a new scratch REPL in your terminal:
```bash
npx @estrattonbailey/scratch index.js
```

Then open your favorite editor:
```bash
vim index.js
```

Updates are transpiled and logged immediately.
